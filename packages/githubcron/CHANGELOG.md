# @snowinch-tools/githubcron

## 0.1.0

### Minor Changes

- Initial release of @snowinch-tools/githubcron

  Features:
  - GitHub Actions powered cron jobs for serverless applications
  - Framework-agnostic with adapters for Next.js, Express, and Fetch API
  - Environment-aware URLs via GitHub Variables
  - Local development worker for testing
  - Secure webhook secret validation
  - Flexible logging with lifecycle callbacks
  - CLI tool for initialization and workflow generation
